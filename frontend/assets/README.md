# Assets

Esta carpeta debe contener las siguientes imágenes:

## Imágenes Requeridas

### 1. icon.png
- **Tamaño:** 1024x1024px
- **Formato:** PNG
- **Descripción:** Ícono principal de la aplicación

### 2. splash.png
- **Tamaño:** 1284x2778px
- **Formato:** PNG
- **Descripción:** Pantalla de inicio (splash screen)

### 3. adaptive-icon.png
- **Tamaño:** 1024x1024px
- **Formato:** PNG con transparencia
- **Descripción:** Ícono adaptativo para Android

### 4. favicon.png (opcional)
- **Tamaño:** 48x48px
- **Formato:** PNG
- **Descripción:** Favicon para versión web

## Cómo crear las imágenes

Puedes usar cualquier herramienta de diseño:
- Canva (https://canva.com)
- Figma (https://figma.com)
- Adobe Photoshop
- GIMP (gratis)

O usar generadores online:
- https://www.appicon.co/
- https://makeappicon.com/

## Colores de la App

- **Primario:** #0da6f2 (azul cliente)
- **Secundario:** #00d4ff (azul claro)
- **Verde (owner):** #4CAF50
- **Morado (admin):** #9C27B0

## Nota Importante

Sin estas imágenes, la app funcionará pero mostrará el ícono/splash por defecto de Expo.
Para generar el APK para producción, estas imágenes son **obligatorias**.
